# Quickstarts

Use [Get Started](../get-started.md) as the canonical first-run guide. It covers installing the extension, reading OTLP files, and sending one OTLP/HTTP log with `curl`.

For the two focused paths:

- [Live Ingest Quickstart](serve.md) - stream OTLP/HTTP into DuckDB or DuckLake.
- [Collector Setup](../setup/collector.md) - configure the OpenTelemetry Collector file exporter.
